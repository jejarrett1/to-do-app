Image Tagging Instructions (Click to Expand/Collapse)

Description: Please help us validate destruction caused by Hurricane Michael. We want to identify any destruction caused to buildings.

  • Select Destoryed [Z] If there is no longer a standing structure on the building footprint.
  • Select Damaged [X] If there is damage present but the building itself is still present.
  • Select Minor/No Damage [C] If the building looks to have suffered no major structural damage.
  • Select Minor/Poor Image [V] If the image is poor quality or you are absolutely unsure.
  • Select Left [Q] Move slide bar to the left to reveal post event imagery.
  • Select Right [W] Move slide bar to the right to reveal pre event imagery.
  • Additional Keyboard Bindings - Back = Backspace Key and  Next/Submit = Spacebar
